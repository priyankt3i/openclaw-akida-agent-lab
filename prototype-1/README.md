# prototype-1

Initial prototype workspace for the spiking attention surrogate experiments on the Akida project.

## Planned structure
- `experiments/`: runnable experiment scripts
- `src/`: reusable modules after patterns stabilize
- `artifacts/`: saved outputs, metrics, and configs

## First goal
Create a deterministic dense attention-like baseline, then compare it against a sparse/spiking surrogate using quality, sparsity, and compute/energy proxy metrics.
